Fred is building a productised client website platform. Not just VFIT — a reusable system for any fitness/wellness/service business.

**The vision:** One command to spin up a fully-featured booking website with admin, payments, notifications, scheduling, analytics, and all integrations. Each client gets their own instance.

**Target market:** Boutique fitness studios, wellness practitioners, PT studios, yoga/pilates studios — small businesses that need bookings + scheduling + payments.

**Key differentiator:** Looks like a $50K custom build but deploys in 60 seconds. No monthly platform fee (just per-transaction Stripe fees).

**How to apply:** Build the template as a comprehensive, production-ready system with every integration baked in. Configuration-driven — swap a config file to brand for a new client. Automate all infrastructure setup.
